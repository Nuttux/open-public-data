Les diagrammes Sankey sont des outils puissants pour visualiser les flux de données. Voici comment les lire et les interpréter.

## Qu'est-ce qu'un diagramme Sankey ?

Un diagramme Sankey est une visualisation qui montre les **flux entre différentes catégories**. La largeur de chaque flux est proportionnelle à sa valeur.

Dans le contexte budgétaire :
- Les **nœuds** représentent les catégories (recettes, dépenses, fonctions)
- Les **liens** représentent les montants qui circulent entre ces catégories

## Comment lire notre Sankey budgétaire

Notre visualisation est organisée en plusieurs niveaux :

### Niveau 1 : Type d'opération
- **Recettes** : l'argent qui entre
- **Dépenses** : l'argent qui sort

### Niveau 2 : Sections
- Fonctionnement (quotidien)
- Investissement (projets)

### Niveau 3 : Chapitres budgétaires
Le détail par nature de recette ou de dépense.

## Astuces d'utilisation

Quelques conseils pour tirer le meilleur parti de notre outil :

1. **Survolez les flux** pour voir les montants exacts
2. **Cliquez sur un nœud** pour filtrer et voir le détail
3. **Changez d'année** pour comparer les évolutions
4. **Utilisez le drill-down** pour explorer les sous-catégories

## Interpréter les proportions

La beauté du Sankey réside dans sa capacité à montrer les proportions d'un coup d'œil :

- Un flux large = un montant important
- Un flux étroit = un montant modeste
- La couleur indique le type (recettes en vert, dépenses en bleu/rouge)

> Les proportions visuelles sont aussi importantes que les chiffres.

## Exemple pratique

Prenons l'exemple des dépenses de personnel :

```
Total dépenses → Section fonctionnement → Chapitre 012 (Personnel)
```

Ce flux montre quelle part du budget total est consacrée aux salaires des agents municipaux.
