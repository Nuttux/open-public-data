Le budget de la Ville de Paris est l'un des plus importants de France. Dans cet article, nous décryptons les grandes lignes du budget 2024.

## Les grandes masses budgétaires

Le budget de Paris se divise en deux grandes sections :

### Section de fonctionnement
C'est le budget du quotidien : salaires des agents, entretien des équipements, subventions aux associations, etc.

```
Recettes de fonctionnement : ~9,5 milliards €
Dépenses de fonctionnement : ~9,3 milliards €
```

### Section d'investissement
Ce sont les projets structurants : construction d'écoles, rénovation de bâtiments, aménagement urbain...

```
Recettes d'investissement : ~2,8 milliards €
Dépenses d'investissement : ~3,2 milliards €
```

## Les principales sources de recettes

Les recettes de Paris proviennent de plusieurs sources :

1. **Fiscalité directe** : taxe foncière, cotisation foncière des entreprises
2. **Dotations de l'État** : dotation globale de fonctionnement
3. **Recettes tarifaires** : cantines, musées, stationnement
4. **Emprunts** : pour financer les investissements

## Les priorités de dépenses

Le budget 2024 met l'accent sur plusieurs priorités :

- **Transition écologique** : 10% du budget d'investissement
- **Éducation** : construction et rénovation d'écoles
- **Logement social** : objectif de 25% de logements sociaux
- **Solidarité** : aide aux plus vulnérables

> Le budget est le reflet des choix politiques d'une collectivité.
